# OmniTrack - Sales Order Management System

## Overview
OmniTrack is a Sales Order Management System built using Flask and MySQL. It helps businesses manage customers, products, inventory, and sales orders from a single platform.

The application provides real-time inventory tracking, order management, customer management, and reporting features.

## Installation

### Clone the Repository

bash
git clone https://github.com/Nandeshb/OmniTrack.git


### Navigate to the Project

bash
cd OmniTrack


### Install Dependencies

bash
pip install -r requirements.txt


### Configure MySQL

Create a MySQL database.
Update the database credentials in app.py.

### Run the Application

bash
python app.py


Open your browser and visit:


http://127.0.0.1:5000
